Simple Crud Demo

This is a site+api for demoing a deployment tool which uses npm and nginx.

After you make changes and commit them, you can run a command which starts up another application instance and switches your server to use that one. With the magic of nginx, the result is a zero-downtime service.

Check out the Github repo for more details.

API

Here are the available API endpoints:

Users

List, create, and update users.

GET /user

Get the list of users.

Parameters:

  • pageSize (integer, optional) Number of results per page.
  • page (integer, optional) The page number.
  • filter (string, optional) Regex used to filter on username.

GET /user/${username}

Get information about a single user.

No parameters.

PUT /user/${username}

Create user for this username.

If the user does not exist, it will be created.

Content type: application/json

Properties:

  • token (string, required) Like a password. Required to update user on later HTTP requests.
  • email (string, required) Email address used to contact user about technical issues.

POST /user/${username}

Update information about this user.

Content type: application/json

Properties:

  • token (string, required) The token used when originally generating the user.
